- `INT-001` 입력 방어의 초기 개발 한도는 보유 항목 최대 500개, 전체 요청 최대 256 KiB, 종목 식별자 최대 64자, 수량은 0보다 크고 `10^15` 이하인 유한 십진수로 한다. 허용 문자 집합은 `OPEN-IMPL-001`의 시장 식별 규칙으로 확정한다.

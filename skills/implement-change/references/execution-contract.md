# Work-packet execution contract

Change only authorized paths and only for the named `WP-*`. Preserve unrelated work and approved baselines. Record every changed path, command, result, skipped check, deviation, assumption, and rollback note. A passing command is evidence, not approval. Never commit, push, merge, deploy, alter approval records, or mark findings resolved unless the packet explicitly authorizes that exact action and the orchestrator still requires a later human gate.
